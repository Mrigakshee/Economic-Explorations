# Economic-Explorations

## Project Overview
This project explores the Human Development Index (HDI), a critical metric used by economists to move beyond GDP and understand the quality of life within a nation. I built this Python tool to analyze how health, education, and income interact to define a nation's development level.

## Case Study: India vs. Norway (2023 Data)
To test the tool, I compared the economic profiles of India and Norway using approximate index values.

| Country | Health Index | Education Index | Income Index | HDI Score |
| :--- | :--- | :--- | :--- | :--- |
| **Norway** | 0.95 | 0.93 | 0.94 | 0.94 |
| **India** | 0.67 | 0.62 | 0.65 | 0.65 |
>*Note: The index values used are normalised representations of life expectancy, mean years of schooling, and GNI per capita, based on standardised UNDP methodology.*

### Economic Insights
*   **The Power of Holistic Measurement:** Using my calculator, we can clearly see the development gap between a highly developed nation like Norway and a developing nation like India. 
*   **Observations:** While GDP often paints a picture of pure wealth, the HDI indices reveal that the gap is not just in income, but fundamentally in health and education outcomes. This tool helps visualize why development requires more than just capital—it requires investment in human capital.

## How to run
1. Ensure Python 3 is installed.
2. Run `python3 hdi_calculator.py` in your terminal.
3. Input the index values (0-1) when prompted.
